# The Obsessed - Bedrock Horror Add-on

## Overview
"The Obsessed" is a horror-themed Minecraft Bedrock Edition add-on inspired by the Java mod of the same name.  
It adds a mysterious stalking entity with a sanity system, camera effects, animations, and sounds to create a suspenseful horror experience.

---

## Features

- **Stalking AI:** The Obsessed entity teleports or vanishes when observed, creating tension.  
- **Sanity System:** Player sanity decreases when near the Obsessed.  
- **Camera Effects:** Screen shakes at low sanity levels.  
- **Animations:** Idle, walking, and twitch animations for realistic movements.  
- **Sounds:** Breathing and whispering audio for atmosphere.
